安装Hexo

Hexo 是一个快速、简洁且高效的博客框架， 使用MarkDown解析文章，通过配合主题使用，可以生成漂亮的静态网页，托管在Github上

在安装hero，需要先安装以下的依赖

  • node.js
  • git

假设你已经安装了上面的一些东西，那么可以直接开始安装hexo。

一条命令就搞定了，如果在国内，将npm替换为cnpm，可以大大提高速度
